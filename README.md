# Professor 📚

Mentor pessoal para alunos do ensino médio.

## Como funciona

1. Solicite um tema (ex: "sistema imunológico", "teorema de pitágoras", "reação de oxirredução")
2. O Professor pesquisa em fontes confiáveis
3. O material é gerado em markdown com hierarquia de temas
4. Conteúdo salvo aqui com todas as fontes citadas

## Matérias

- [Matemática](./matematica)
- [Biologia](./biologia)
- [Física](./fisica)
- [Química](./quimica)
- [História](./historia)
- [Português](./portugues)
- [Geografia](./geografia)

## Formato do conteúdo

Cada matéria segue uma estrutura hierárquica natural:
```
matéria > tópico > subtópico > conceito específico
```

Todo material inclui:
- Introdução com contexto prático
- Conceitos-chave explicados
- Exemplos do dia a dia
- Fontes com links

## Contribuir

Este é um repositório vivo, atualizado conforme os alunos solicitam novos temas.
